# Evaluation de bibCheck

Pour évaluer bibCheck :

- Installer les dépendances `requirements.txt` (nous préconisons python3.9)
- Utiliser le corpus disponible sur Ortolang `bibCheck`
- obtenir les résultats en mettant en entrée standard le corpus de bibCheck. Par exemple : `python3.9 evaluate-bibcheck.py < corpus_bibCheck.jsonl`
